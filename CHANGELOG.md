# 更新日志

本文件用于按“日期”记录项目功能和行为变化。

## 统计

- 累计记录天数：2

## 记录规则

- 每次提交前必须更新对应日期的记录。
- 记录格式为“日期 + 日修改”。
- 同一天多次提交时，统一归并到同一条“日修改”下持续追加。
- 先写用户可感知变化，再写内部重构。
- 必须包含影响文件列表。

## 2026-04-16

#### 新增

- 新增流程中 Alt + Right 重置能力，可在测量过程中快速回到第一步。
- 新增主窗口“首页/设置页”单窗口切换，设置项从首页逻辑中解耦。
- 新增 Alt + Q 触发窗口设置（0.3 / 0.5 / 0.8 秒）并支持本地持久化。
- 新增中英双语切换（默认英文），并覆盖主界面及测量悬浮提示文本。
- 新增热键状态机与设置存储模块，拆分核心流程逻辑，提升可维护性。
- 新增开发者流程说明，明确启动、退出、防误触、重置的状态行为。
- 新增并后续移除自动化发布脚本 `scripts/release.ps1`，最终回归轻量提交流程。

#### 修复

- 修复流程中 Alt + Q 退出体感滞后问题，关键等待段可及时消费退出请求。
- 修复流程期间偶发“未响应”现象，等待循环中持续泵主窗口事件。
- 修复流程中 Alt + Q 退出判定范围过大的问题：
  - 1.5 秒防误触仅在“进入测量后的前 1.5 秒”生效；
  - 超过该窗口后 Alt + Q 直接退出，不再延迟判定。

#### 优化

- 优化热键时序判定，降低语音 Alt 键场景下误触概率。
- 优化测量流程控制，统一 reset / exit / done 状态分支，行为更可预测。
- 优化悬浮提示窗口实现，避免额外创建 Tk 根窗口。
- 优化文案与设置管理，抽离 i18n 文案表和 settings 存储逻辑，减少 `main.py` 耦合。
- 优化仓库协作规范与文档，补充开发与维护说明。

## 2026-04-17

#### 新增

- 新增 Windows 全局快捷键轮询方案（Alt/Q），程序不在前台时也可触发测量流程。
- 新增设置文件路径迁移到 AppData：`%APPDATA%/MortarAid/settings.json`，并保留旧路径读取兜底。
- 新增 PyInstaller spec 运行库收集逻辑，自动附带 Tk/Conda 常见依赖 DLL。
- 新增 Zip 分发指引与打包命令（`Compress-Archive`）。
- 新增英文文档 `README.en.md`，与中文文档形成双向入口。

#### 修复

- 修复游戏前台运行时本工具在后台无法识别快捷键的问题。
- 修复中文设置页在默认窗口尺寸下的文本显示不完整问题（缩小中文 UI 字体）。
- 修复打包后 `_tkinter` 运行时缺失导致 EXE 启动失败的问题。
- 修复 onedir 打包场景下帮助页图片路径解析，避免发布版说明页图片丢失。

#### 优化

- 优化设置写入流程：保存前自动创建配置目录，降低首次运行写入失败风险。
- 优化打包与运行一致性：README 更新为基于 spec 的一键打包方式。
- 优化发布流程：推荐 onedir + zip 分发，降低下载体积并提升启动稳定性。
- 优化打包体积：PyInstaller 显式排除常见无用大模块（numpy/pandas/matplotlib/scipy、Qt、Jupyter）。
- 优化资源打包：帮助页所需图片与图标纳入发布产物。
- 优化 README 导航：中文 README 顶部增加英文入口，默认中文为主。
- 优化 README 运行指引：补充“快捷键无法触发时以管理员身份运行”排障提示。
