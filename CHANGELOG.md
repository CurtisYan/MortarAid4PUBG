# 更新日志

本文件用于记录项目每次功能和行为变化。

## 统计

- 累计更新次数：4

## 记录规则

- 每次提交前必须新增一条记录。
- 记录格式为“日期 + 当日第 N 次更新（更新 #编号）”。
- 先写用户可感知变化，再写内部重构。
- 必须包含影响文件列表。

## 2026-04-16

### 第 1 次更新（更新 #001）

#### 新增

- 新增 Alt + Right 重置功能：测量过程中可随时重置并回到第一步。
- 主界面控制说明新增重置快捷键提示。

#### 优化

- 测量流程改为可重启循环，重置后不需要退出程序重开。
- 采点等待逻辑支持重置中断，重置响应更快。

#### 影响文件

- main.py

#### 优化

- README 增加自动化发布命令和参数说明，降低手工维护成本。

#### 影响文件

- scripts/release.ps1
- README.md
- CHANGELOG.md

### 第 3 次更新（更新 #003）

- CHANGELOG 采用“当日第 N 次更新（更新 #XXX）”规则，支持同日多次提交。

### 第 4 次更新（更新 #004）

#### 新增

- 主窗口新增菜单栏：Settings -> Language，可在 English/Chinese 间切换。
- 软件默认英文显示；切换中文后，主界面和测量悬浮提示同步切换为中文。

#### 优化

- 多语言文案统一由 i18n 字典按 key 管理，减少散落硬编码文本。
- 抽象测量流程中的重复提示与双点采集逻辑，降低重复代码成本。
- 预解析 Alt 修饰键并复用，减少采点热路径中的重复字符串拆分开销。
- 悬浮提示窗口改为 Toplevel，避免额外创建 Tk 根窗口。

#### 影响文件

- main.py
- CHANGELOG.md
